# 13. 选择决策矩阵（Decision Matrix）

| 决策因素 | 行为统计 | 行为预测 | 行为模拟 |
| :--- | :--- | :--- | :--- |
| 成本投入 | 低 | 中 | 高 |
| 技术难度 | 低 | 中 | 高 |
| 时效要求 | 毫秒级 | 毫秒级 | 秒至分钟 |
| 扩展性 | 极强 | 强 | 中等 |
| 价值浓度 | 基础 | 中等 | 高 |

- 为高风险、高价值、低历史数据支撑的议题优先考虑模拟范式；日常、成熟场景仍以统计与预测为主。
- 推荐建立“选择流程”：定义问题→评估范式适配度→配置资源→设定验证指标→执行与复盘。
